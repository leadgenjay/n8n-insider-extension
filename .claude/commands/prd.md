# Create PRD.md

Feature/Project: $ARGUMENTS

Create a PRD.md file with:
1. **Overview**: What we're building and why
2. **Goals**: Success metrics and objectives
3. **User Stories**: As a [user], I want [goal] so that [benefit]
4. **Technical Requirements**: Architecture decisions for Next.js/Supabase/n8n
5. **Data Model**: Supabase tables and relationships
6. **API Endpoints**: n8n webhooks needed
7. **UI Components**: Pages and components to build
8. **Implementation Plan**: Ordered steps to execute
9. **Open Questions**: Things to clarify

Save to `PRD.md` in the project root.
